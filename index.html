Formulário com consumo de API's

Endereço residencial


O formulário foi aprimorado para oferecer uma experiência mais eficiente e interativa aos usuários. Para isso, foi integrado duas APIs poderosas: o ViaCEP, que preenche automaticamente os campos de endereço ao inserir o CEP, e o SheetMonkey, que envia os dados para o Google Drive, garantindo o armazenamento seguro das informações coletadas. Além disso, implementamos validações em JavaScript para assegurar a precisão dos dados inseridos e animações em CSS para tornar o processo de preenchimento mais agradável e fluído. Com essas melhorias, buscamos oferecer ao usuários uma experiência mais intuitiva e simplificada.
